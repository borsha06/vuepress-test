## Local Development

To try it out locally, clone this repo and generate a static site

```bash
git clone https://github.com/borsha06/vuepress-test
cd vuepress-test
npm install
vuepress dev

npm run build
```

For more information, see [VuePress Docs](https://vuepress.vuejs.org)
